## Actividad con el log in
 
Este proyecto utilizo tecnologías Laravel, Breeze y Socialite. 

Breeze se utilizo para realizar el inicio de sesión mediante un correo y contraseña que se guarda en la BD.

Después se realizo una modificacion en un campo de los usuarios en la BD para que pudiera registrase su id de google.

También se modifico un archivo ubicado en la carpeta de system 32 para agregar el host al sistema. Al igual que el archivo .env en el proyecto.
 
Mediante el uso de Google Cloud Console se utilizó OAuth 2.0, así se pudo conectar al proyecto y autenticarse con una cuenta de Google.
